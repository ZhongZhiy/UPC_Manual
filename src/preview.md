# 前言
本书是UPC生活指南, 它可能不会比校/院发的各种指南全面, 官方, 但是它记录的是哪些不在官方文书中的 **生活经验**. 对于每个人(也许大多数)来说这都是第一次上大学, 过往的大多数经验在这里都不再适用, 每个人不得不走过相同的路, 踩过相同的坑, 上相同的当, 最后成长为学校里的"老生", 然后离开学校, 然后又是一批新人到来... 反反复复, 浪费一代又一代人的时间. 所以我想, 如果能分享老生的经验, 磨平一部分信息差, 或许可以为很多人节约大量的试错成本.
此外, 涉及 **大学之后的事情**, 大部分同学大概分成两派 -- **读研和工作**, 如果没有及时意识到自己的出路再开始准备可能就已经迟了, 所以本书也会在这两个方面着色, 为同学们提供相比网络其他资源 **更加接近UPC本土化** 的指南. [上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual/li-zhi-pian/huan-ying-lai-dao-shang-hai-jiao-tong-da-xue)中说:
> 国内绝大部分大学的本科教学，不是濒临崩溃，而是早已崩溃。

如果你还是维持着高中思维, **跟着学校安排走**, **听辅导员的就是了**, 那么到大四毕业的时候, 你很可能浪费了4年青春, 绩点也在那瞬间失去意义, 仅仅学到一些陈腐的知识, 何况在考完就忘的情况下, 连那一点的知识可能也不存在了, 那么如何能应对企业要求, 或如何让导师选择你, 所以我 **强烈推荐你阅读[上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual/li-zhi-pian/huan-ying-lai-dao-shang-hai-jiao-tong-da-xue)**, 它的[立志篇](https://survivesjtu.gitbook.io/survivesjtumanual/li-zhi-pian/huan-ying-lai-dao-shang-hai-jiao-tong-da-xue)的认知一定会让你重新思考应该如何度过在UPC的大学时光.

这本书刚开始仅仅是笔者的个人经验之谈, 所以内容难免片面, 如果你愿意分享你的经验, 欢迎投稿, 投稿参考[README](./README.md).
