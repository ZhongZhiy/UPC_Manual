# README
UPC Manual 是笔者阅读了[上海交通大学生存手册](https://survivesjtu.gitbook.io/survivesjtumanual)后想到, "或许也可以做一个UPC生存手册, 写下在UPC生活的经验分享初来乍到的学子", 在这种想法下, 我果然拖延了半年:) , 在最近(26/5/6), 但我在次阅读到上交生存手册, 这种想法再次袭来, 于是便有了这个UPC小册子最初的样子. 这本电子书使用[mdbook](https://rust-lang.github.io/mdBook/)制作, 内容托管在[GitHub仓库中](https://github.com/ZhongZhiy/UPC_Manual) , 为了方便国内访问, 部署在[netlify](https://app.netlify.com/projects/upc-manual/overview)上. 

目前本书还在建设阶段, 暂定大纲如下:
1. 学习篇
2. 生活篇
3. 保研篇
4. 工作篇
5. 相关资源

## 投稿
欢迎在校和已经毕业的同学投稿, 投稿有以下两个方法:
1. 阅读[mdbook文档](https://hellowac.github.io/mdbook-doc-zh/zh-cn/index.html), 给[UPC_Manual仓库](https://github.com/ZhongZhiy/UPC_Manual)提pr
2. 发送markdown文档到邮箱`sunupper.shine@gmail.com`

## 提issue
如果你觉得本书内容有任何需要修改的地方, 请在[本书仓库](https://github.com/ZhongZhiy/UPC_Manual)中提issue, 或者发送问题到邮箱 `sunupper.shine@gmail.com`

如果你觉得本书对你有帮助的话, 请给[本书仓库](https://github.com/ZhongZhiy/UPC_Manual)一个star, 或者分享给你的伙伴, every action matters.
